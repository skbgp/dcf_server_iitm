1. Essential File I/O (CSV Parsing)

How to quickly read comma-separated files or handle testcase inputs natively.

#include <iostream>
#include <fstream>
#include <sstream>
#include <string>

int main() {
    std::ifstream file("input.csv");
    std::string line;

    // Read line by line until EOF
    while (std::getline(file, line)) {
        std::stringstream ss(line);
        std::string token;
        
        // Split current line by commas
        while (std::getline(ss, token, ',')) {
            std::cout << token << " ";
        }
        std::cout << "\n";
    }
    return 0;
}

2. Vectors & Standard Template Library (STL)

Common operations on dynamically allocated arrays.

#include <vector>
#include <algorithm>

std::vector<int> v = {4, 2, 5, 1, 3};

// Push to back O(1)
v.push_back(6);

// Sort ascending O(N log N)
std::sort(v.begin(), v.end());

// Sort descending
std::sort(v.begin(), v.end(), std::greater<int>());

// Iterating modern way
for(int x : v) {
    std::cout << x;
}

4. Fast I/O Configuration

If you have tight time limits, disabling C++ and C stream synchronization can drastically decrease `cin` and `cout` time complexity operations.

int main() {
    // Add these two lines at the top of main()
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);
    
    // ... your logic
}
